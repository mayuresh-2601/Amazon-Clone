### STEP 1 – PROJECT OVERVIEW (0:30 – 1:00)

This project is a **frontend-only clone of the Amazon homepage**.
The main goal was to understand how **real-world e-commerce UIs are structured**, how layouts are built, and how **user interactions** are handled.

The project focuses on:

* Clean UI structure
* Responsive design
* Basic JavaScript interactivity
* Accessibility best practices

---

### STEP 2 – NAVIGATION BAR (1:00 – 1:40)

👉 *Scroll to top, point to navbar*

This is the **navigation bar**, which is the most important part of any e-commerce website.

Here we have:

* Amazon logo
* Delivery location
* Search bar
* Account and orders section
* Cart icon

I built this using **Flexbox**, so all elements stay properly aligned across screen sizes.
When I scroll down, you can see that the navbar becomes **sticky**, which is handled using JavaScript.

This improves user experience and matches real production websites.

---

### STEP 3 – SEARCH FUNCTIONALITY (1:40 – 2:10)

👉 *Type something in search box*

The search bar is interactive.

When I type a product name and click the search icon or press Enter, JavaScript captures the input and simulates a search action.

If the input is empty, it shows a validation message.
This demonstrates **event handling, DOM manipulation, and input validation**.
