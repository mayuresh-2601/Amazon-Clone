# 🛒 Amazon Clone – Frontend E-Commerce UI Project

![Amazon Clone Banner](asset/amazon_logo.png)

## 📌 Project Overview

This project is a fully responsive Amazon homepage clone built using **HTML, CSS, and JavaScript**.

It replicates the core UI layout and interactive features of Amazon’s landing page, including navigation, product sections, search functionality, cart simulation, and responsive design behavior.

The goal of this project was to strengthen frontend development skills and understand real-world e-commerce UI structure.

> ⚠ This project is for educational purposes only and is not affiliated with Amazon.

## 🧠 JavaScript Functionalities Implemented

### 🔍 Search Feature
- Detects input value
- Triggers search on button click
- Allows search using Enter key
- Displays alert simulation

### 🛒 Cart Simulation
- Increments cart count when clicking product links
- Dynamically updates cart UI

### 📌 Sticky Navbar
- Detects scroll position
- Fixes navbar to top dynamically

### ⬆ Back to Top
- Smooth scrolling effect using `window.scrollTo`

### 📱 Mobile Menu Toggle
- Shows/hides navigation items on smaller screens
